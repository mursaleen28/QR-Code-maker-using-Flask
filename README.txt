QR Code Maker – How to Use

Description:
Type any text or link → get a QR code → click to download.
No database. Built using Flask. Works offline on desktop browser. Can also run on mobile (through same Wi-Fi).

How to Run:

1. Extract the ZIP.
2. Open terminal inside the folder.
3. Run this command:
   python app.py

4. After running, a URL will appear like:
    Running on http://127.0.0.1:5000
   → Click on it or open in your browser.

5. To Access on Phone (Same Wi-Fi):
   a. Open CMD on your laptop and type:
      ipconfig
   b. Copy your laptop’s IP address (e.g., 192.168.x.x)
   c. On your phone browser, type:
      http://192.168.x.x:5000
 
 Offline Use:
- Desktop: Open once with internet and works offline too.
- Phone: Must be on same Wi-Fi. Offline mode on mobile may not fully cache due to browser settings.
